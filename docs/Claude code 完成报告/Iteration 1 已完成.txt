  2. 新增接口契约

  POST /api/v1/reports/assistant/messages

  - 请求：ReportAssistantMessageRequest（message + conversation_context + client_request_id）
  - 响应：ReportAssistantMessageResponse（status + intent + report_id + answer + assumptions + evidence + suggested_follow_ups）
  - 权限：ensure_management_user（学生拒绝）+ catalog 角色过滤
  - 降级：REPORT_ASSISTANT_LLM_ENABLED=false 时使用关键词路由，不依赖 LLM

  9. 已知限制

  1. 无 LLM 环境未端到端测试：关键词降级模式经过测试；LLM Structured Output 模式需要 REPORT_ASSISTANT_LLM_ENABLED=true + 有效 API Key
  2. 会话上下文不持久化：前端需要每次请求传回 conversation_context
  3. 多 worker 不支持：无共享会话存储；Iteration 2 需要 context.py 独立实现
  4. 心理报告权限：仅 admin/manager/team_leader 可访问
  5. 报告同步生成：generate_report() 同步调用可能阻塞请求线程 3-10s
